== Overview

Clicky is a CLI kanban board designed for human-agent collaboration. It provides a simple yet powerful interface for managing tasks through command-line interactions, making it ideal for automation and scripting while remaining user-friendly for manual use.

== Features

* *CLI-First Design*: All operations available through intuitive commands
* *Human-Agent Collaboration*: Designed for both manual use and automation
* *Persistent Storage*: JSON-based storage for easy inspection and versioning
* *Clean Architecture*: Maintainable and testable codebase
* *Extensible*: Ready for future TUI and additional features
